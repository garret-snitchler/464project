AI Chatbot

This work was done for a corporation, so I can't name too many specifics. However, using Microsoft Azure to connect to an OpenAI subscription, myself and my team were able to create an in-house chatbot powered by generative AI. This chatbot was fed on internal documentation to make it easier for new and old employees alike to quickly reference protocols and other information that can otherwise require a lot of digging. Additionally, the tool was capable of citing the documents it was getting its information from and could have new information sources added to it through an administrator view.

Automatic Email Notification System

This work was done for a corporation, so I can't name too many specifics. However, I led a team that created an automatic email notification system. This system used Snowflake to access the company's database. By using Snowflake to register changes in the data, we were able to then connect to the company's in-house email messaging API to then send notifications to employees about when certain tasks they were waiting on had been completed. Also with Snowflake, we would find the email address of the specific employee that the task was relevant to to avoid spamming an entire office or role over something only one person would care about. This project is ongoing and we are currently working to get this system set up on Kubernetes.

Tic Tac Toe in Godot

This was some solo game development work I did as a personal project. My goal was to get more comfortable with game development and Godot as a game engine. I used the familiar game of Tic Tac Toe as a blueprint and then began work on all of the different work that would go into making it as functional on a computer as it would be on a piece of paper. This involved getting assets into the game, learning gdscript, scene management, and more. After initial functionality was established, I tried to put as much quality of life into the game as possible, though ultimately never went as far as setting up multiplayer. I ended the project by teaching myself how to export a game into an executable format in order to make things more user friendly.
